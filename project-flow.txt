Tableli Platform — Complete End-to-End Project Flow (বাংলা ভার্সন)

আমি আপনার সম্পূর্ণ project structure এবং এখন পর্যন্ত implement করা modules বিশ্লেষণ করেছি। নিচে Tableli Platform-এর সম্পূর্ণ End-to-End Flow দেওয়া হলো, যেখানে Service Provider (Chef), Consumer (Customer), এবং Platform Admin — তিনটি side-এর workflow ব্যাখ্যা করা হয়েছে।

1. Service Provider Journey (Chef)
A. Registration & Setup
Authentication (/auth)

Provider প্রথমে registration করবে এবং chef role নির্বাচন করবে।

Features
Email verification
OTP verification
Secure authentication
JWT-based login system

Profile Setup (/profileSetup)

বুকিং রিসিভ করার আগে provider-কে profile setup করতে হবে।

Profile তথ্য
Full Name & Display Name
About Me
Location (City, Country)
Cuisine Specialties & Category (yacht, villa, event, personal)
Menu Details (Starting price, Sample Menu)
Visual Portfolio (Dish images, Event highlights)
Identity Verification (CV, Government ID, Food Safety Certificate)

Subscription Selection (/subscription)

Provider membership plan নির্বাচন করবে (যদি থাকে)।

B. Menu & Service Management
Menu Creation (/menuCreate)

Provider তার menu বা dishes create করবে।

Example Cuisines/Services
French Gastronomy
Italian Classic
Sushi Course
Bespoke Menu

Service Details
প্রতিটি menu-এর জন্য:

Menu Title
Price per Person
Courses
Images

Availability Management
Provider available dates এবং service windows (Dinner, Brunch, Lunch) setup করবে।

C. Booking & Client Interaction
Incoming Bookings (/booking-chef)

Customer booking করলে provider notification পাবে।

Notification System (/notification)
Provider real-time notification পাবে:

New Booking Request
Booking Cancelled
Review Added

Booking Approval System
Provider pending bookings দেখতে পারবে এবং:

Action	Result
Confirm	Booking Confirmed
Cancel	Booking Cancelled

Review Management (/review)
Customer review দিলে provider তা দেখতে পারবে।

Review Data
Rating
Comment
Customer Feedback

D. Performance Tracking
Dashboard (/dashboard)

Provider business analytics দেখতে পারবে।

Dashboard Metrics
Analytics	Description
Revenue Charts	আয়
Total Bookings	মোট বুকিং
Profile Views	প্রোফাইল ভিউ

2. Consumer Journey (Customer)
A. Chef Discovery
Search System (/search)

Customer nearby chef বা specific cuisine search করবে।

Search Priority Logic
System verified এবং highly rated chef-দের priority দেবে।

Provider Profile View
Customer দেখতে পারবে:

Chef's Portfolio
Menu & Pricing
Ratings & Reviews
Available Dates

B. Booking a Chef
Availability Check
Customer নির্দিষ্ট date-এ chef-এর availability চেক করবে।

Booking Creation (/booking-chef)
Customer guest count, date, arrival time, এবং location দিয়ে booking request submit করবে। সিস্টেম স্বয়ংক্রিয়ভাবে Subtotal, Concierge Service Fee (10%), এবং Estimated Taxes (8%) ক্যালকুলেট করে Total Amount জেনারেট করবে। 
*বুকিং অবশ্যই ২৪ ঘণ্টা আগে করতে হবে এবং শেফের মিনিমাম বুকিং অ্যামাউন্ট রিচ করতে হবে।*

Booking Notifications
Customer notification পাবে যখন provider:

booking confirm করবে
booking cancel করবে

C. Post-Service Experience
Reviews & Ratings (/review)
Service complete হওয়ার পরে customer review দিতে পারবে।

3. Platform Admin Journey (Super Admin)
Admin Dashboard (/dashboard)

Admin পুরো platform monitor করতে পারবে।

Admin Metrics
Metric	Purpose
Total Revenue	Platform Income
Total Users	User Count
Active Chefs	Chef Tracking

Chef Profile Verification
Admin শেফদের ভেরিফিকেশন ডকুমেন্ট চেক করে প্রোফাইল approve করবেন। Approve না হলে শেফ বুকিং পাবেন না।

Moderation System (/report)
Admin reported content manage করবে।

4. Technical Infrastructure (Behind the Scenes)
Database Layer (src/config/database)

MongoDB + Mongoose ব্যবহার করা হয়েছে।

Core Responsibilities
System	Responsibility
MongoDB	Data Storage
Mongoose	Relationship Handling
JWT	Authentication
Middleware	Access Control

Authentication Middleware (src/helper/middlewares)
এই layer নিশ্চিত করে:

User শুধু নিজের data access করতে পারবে
Unauthorized request block হবে
Chef specific route শুধুমাত্র chef access করতে পারবে

Payment Gateway (src/payment)
Stripe ব্যবহার করা হয়েছে পেমেন্ট প্রসেসিংয়ের জন্য।

Real-time Communication (src/socket)
Socket system ব্যবহার করা হয়েছে instant notifications এবং live messaging-এর জন্য।

Overall System Flow
Customer
   ↓
Search Chef
   ↓
Book Chef (with Date, Guests & Invoice Auto-calc)
   ↓
Chef Confirms Booking
   ↓
Notification System
   ↓
Service Completed
   ↓
Review & Rating
   ↓
Analytics Updated
